MCP1755 56c

MIC5504 12c

Current:

ATMEL ATtiny85

```
3.3V @ 8MHz = 3mA
5V @ 8MHz = 5mA
< 1MHz = 1mA
```

HM-10

```
3.3V = 50mA //Active state 8.5mA, Sleep state 50-200uA
```

OSRAM LRTB GFTG

```
Red = 5mA min, 40mA max
Green = 5mA min, 50mA max
Blue = 5mA min, 50mA max
```
